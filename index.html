Setlist

Describe a moment in plain words. Setlist reads it as sound — tempo, energy, brightness, air — and finds the songs in your library that already live there.

Setup

One-time. Create an app in the Spotify dashboard, register the redirect URI below exactly as written, then paste the Client ID here. Nothing leaves your browser.

Redirect URI to register

Open the Spotify developer dashboard → Create app → set Redirect URI → choose Web API → save, then copy the Client ID.

Since February 2026, Development Mode apps need the owner to have Spotify Premium, allow one Client ID per developer, and cap you at five authorised users. If login fails with a 403, add your own account under Users and access in the dashboard.
